# DSA Practice

A collection of small Python scripts for practising Data Structures & Algorithms fundamentals.

## Files

| File | Description |
|------|-------------|
| `count_of_char.py` | Count the frequency of each character in a string |
| `reverse_string.py` | Reverse a string using a manual loop |
| `reverse_string_input.py` | Reverse a string with live user input (type `stop` to quit) |
